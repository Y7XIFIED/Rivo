## Overview
Rivo is a static, motion-led landing page built for brand storytelling.  
It combines clean structure with interactive behavior using only native web technologies, making it easy to run, audit, and customize.
It is designed to stay visually strong while remaining simple to maintain.

Live site: [https://rivoooo.vercel.app](https://rivoooo.vercel.app)

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla, ES6+)

## Features
- Cinematic one-page layout optimized for visual narrative flow
- Smooth motion interactions and reveal-on-scroll effects
- Section-based navigation with active state tracking
- Mobile-friendly menu behavior and responsive layout handling
- Lightweight architecture with no framework or build dependency
- Fast local iteration with direct file-based editing
- Deployable to any static hosting platform

## Installation
1. Clone the repository.

```bash
git clone https://github.com/Y7XIFIED/Rivo.git
```

2. Move into the project directory.

```bash
cd Rivo
```

3. Serve the site locally (recommended).

```bash
npx serve .
```

4. Open the local URL printed in the terminal.
